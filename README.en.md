# dockit4j

#### Description
dockit4j 是一个面向 Spring Boot 应用的 OpenAPI 3 接口文档增强解决方案，致力于提供比 Swagger-UI 更现代、可定制性更强的 UI 展示体验。
它深度集成 SpringDoc，支持接口分组、文档聚合、元信息扩展等功能，搭配 dockit4j-ui 前端项目，为开发者提供完整的一体化 API 文档管理与调试平台。
